Name
====
CSVデータを取得、特定のセルに代入

## Overview
別フォルダに入っているCSVファイルをすべて読み込み

欲しい情報のみを切り出して、特定のセルに代入する。

## Demo
![CSV特定のセル](https://user-images.githubusercontent.com/43311555/61620217-64adaf00-acab-11e9-8d45-811de1649598.gif)

## Usage
パスを設定し直す必要あり。

特定のセルに代入するのであれば、セルの位置を変更する必要有り。

## Catution
WindowsOSのみ確認
